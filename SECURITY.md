# Security Policy

myPad is a local-only notes app.

## Reporting Security Issues

If you find a security issue, please open a private report through GitHub's
security advisory feature if it is enabled on the repository.

If security advisories are not enabled yet, open an issue with a minimal
description and avoid posting sensitive exploit details publicly.

## Current Scope

- Local scratch-note storage.
- Local app bundle build and install scripts.
- No network sync.
- No telemetry.
